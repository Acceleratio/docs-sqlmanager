SQLDocKit supports the following SQL Server versions:

* SQL Server 2005  
* SQL Server 2008
* SQL Server 2008 R2
* SQL Server 2012
* SQL Server 2014
* SQL Server 2016

>*__Please note!__ We fully support all the available editions of the abovementioned  SQL Server versions, including Express and Developer.*
